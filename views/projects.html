@theRealShayon

This was a Data Structures (CSCI 2270) project that a partner, Nick Hanselman and I collaborated to create. the objective was to use any data structure we liked in a creative way. And boy did we get creative. We used a graph data structure as a Markov Chain to generate sentences based off things our data structures professor said in lecture. First, the program would read a text file in and would store each unique word in the graph. Probabilities of how often each word occurred after it's parent were calculated and used to traverse through the graph using a sudo-random method of finding a path and generating the sentence. It would then output the finished sentence to a file and a script my partner wrote in python using the Twitter API would take the file, read it and Tweet it to the Twitter account we had set up.
